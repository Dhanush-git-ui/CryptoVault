Overview

This project implements a hybrid cryptographic system combining Modified Diffie-Hellman (MDH) and RSA algorithms. The system enhances security by integrating key exchange with encryption mechanisms.

Unlike traditional Diffie-Hellman, this approach introduces authentication during key exchange and prevents man-in-the-middle attacks. The RSA algorithm then uses securely generated prime numbers from MDH for encryption.

Algorithm Steps
1. User A generates prime P and sends Pn = P × 2
2. User B generates Q and sends Qn = P + Q
3. User A recovers Q from Qn
4. Both users generate public keys (PubA, PubB)
5. Secret keys are computed independently
6. If keys match → communication continues
7. RSA uses generated P and Q for encryption

This ensures both authentication and confidentiality, making the system resistant to impersonation and interception attacks :contentReference[oaicite:1]{index=1}.
